Бамбуковая столешница и беспроводная зарядка

Экологичная и долговечная поверхность

Эргономичная столешница из бамбука даёт лёгкое и приятное ощущения естественности на рабочем месте. Это не просто стильное и экологичное решение — бамбук в два раза прочнее обычного дерева!

Встроенная беспроводная зарядка позволяет насладиться отсутствием проводов и устраняет необходимость подключать и отключать кабель для зарядки. Она подходит для большинства устройств с беспроводной зарядкой.

Интеллектуальная панель управления с разъёмами USB

Соответствие вашему уровню

Программируемые настройки высоты: стол идеален для нескольких пользователей, ведь он позволяет легко установить нужную высоту одним нажатием клавиши.

Разъёмы USB для зарядки: встроенные разъёмы USB для зарядки (один USB Type-A, один Type-C) совместимы с широким ассортиментом электронных устройств. Они устраняют беспорядок проводов под вашим рабочим столом.

Практичный деревянный брусок

Используйте с кронштейном для монитора

Стол FoxGear FG-ED-55 поставляется с практичным деревянным бруском. Он заполняет зазор под столешницей и обеспечивает прочную основу для простого крепления кронштейна для монитора.
